# Ayurva - Multilingual AI Healthcare Chatbot

Ayurva is a conversational AI designed to educate users about preventive healthcare, disease symptoms, and vaccination schedules, with a focus on rural Indian communities.

## Project Structure
- `backend/`: Node.js/Express server
- `frontend/`: React/Tailwind mobile-first UI
- `ml-models/`: NLP and Medical Q&A models
- `docs/`: Technical documentation
